# 백준 문제풀이 (12월 22일)

## 2839번 설탕 배달

```python
n = int(input())

salt = [3, 5]

dp = [5000] * (n + 1)

dp[0] = 0
for i in range(2):
    for j in range(salt[i], n + 1):
        if dp[j - salt[i]] != 5000:
            dp[j] = min(dp[j], dp[j - salt[i]] + 1)


if dp[n] == 5000:
    print(-1)
else:
    print(dp[n])
```

풀이법은 책의 효율적인 화폐 구성(p.226) 과 같다.

## 1463번 1로 만들기

```python
x = int(input())

dp = [0] * 1000001

for i in range(2, x + 1):
    dp[i] = dp[i - 1] + 1
    if i % 2 == 0:
        dp[i] = min(dp[i], dp[i // 2] + 1)
    if i % 3 == 0:
        dp[i] = min(dp[i], dp[i // 3] + 1)

print(dp[x])
```

마찬가지로 책과 비슷하다.
